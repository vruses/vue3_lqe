## 2. 项目说明：

### 2.1 目的：
基于VUE框架模块化编程配合WebGIS开发

### 2.2 主要功能

1. 前端项目搭建
2. 登录，路由鉴权
3. 页面适配移动端
4. 丰富的地图api，主要表现在矢量图层的绘制、保存与渲染

### 2.3 主要技术

> 地图框架：OpenLayers

> 前端页面：Vue3 + ElementPlus + TailWind + Vue3OpenLayers + TypeScript

> 数据请求及拦截：Axios

> 仓库与路由：Pinia + VueRouter
